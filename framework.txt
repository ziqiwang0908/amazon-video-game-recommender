Project Framework: Video Game Recommendation System

1. Project Goal

Build an end-to-end recommender system for video games using Amazon Review Data 2018. The project addresses two recommender-system tasks:
- Rating prediction for held-out user-item ratings.
- Top-10 recommendation for unseen games.

The final deliverables include:
- Python recommendation pipeline
- Evaluation results and result plots
- Interactive Next.js demo
- LaTeX report
- Beamer presentation slides

2. Dataset

Main files:
- Video_Games_5.json.gz
- meta_Video_Games.json.gz

Expected default data layout:
- data/raw/
- data/processed/
- data/results/

For large local runs, set MDM_DATA_ROOT to an external data directory. The public repository should not include raw data, processed data, or large generated outputs.

4. Pipeline

Raw review and metadata files
-> Preprocessing and metadata cleaning
-> Per-user 80/20 train-test split
-> Mean baseline prediction
-> Item-item collaborative filtering
-> Top-10 recommendation generation
-> Evaluation
-> Demo JSON export
-> Report and slides

5. Models

Baseline:
- Mean-based predictor using user mean, item mean, and global mean fallback.

Main model:
- Item-item collaborative filtering.
- Cosine similarity between item rating vectors.
- Weighted average prediction using similar items rated by the user.
- Top-10 recommendations generated from unseen candidate items.

Ranking baseline:
- Popularity-based Top-10 recommendation.

6. Evaluation Metrics

Rating prediction:
- MAE
- RMSE

Top-10 recommendation:
- Precision@10
- Recall@10
- F1@10
- NDCG@10
